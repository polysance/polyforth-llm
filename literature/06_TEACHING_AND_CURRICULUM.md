# 06 - Teaching and Curriculum

## Teaching objective

Provide source-grounded explanations that help learners progress from beginner to intermediate polyForth fluency.

## Pedagogical modes in this repo

1. Explain mode
2. Socratic mode
3. Quiz mode

Implementation reference:

- `polyforth_llm/teaching.py`

## Quality characteristics for tutor answers

1. Accurate to source context
2. Explicit citations
3. Appropriate difficulty level
4. Clear statement when context is missing

## Curriculum progression model

1. Foundations (stack model, core words)
2. Control flow and factoring
3. Memory/data and debugging workflows

## Feedback loop

1. Capture weak answers.
2. Identify whether issue is retrieval, prompt, or model.
3. Apply targeted change.
4. Re-run benchmark and compare.
